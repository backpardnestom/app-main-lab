# app-main-lab

用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立编写的 HTML 页面资源，适用于快速预览、分发或存档静态页面。每个页面均以独立的文件形式存在，不依赖特定后端服务或框架，可直接在浏览器中打开运行。

## 目录说明

```
app-main-lab/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面所需的公共资源（如样式、脚本、图片等）
└── README.md       # 本说明文件
```

- `pages/`：每个子文件夹或文件对应一个完整的 HTML 页面，文件名或文件夹名建议体现页面功能或内容。
- `assets/`：用于存放多个页面可能共用的静态资源，减少重复。如有页面专属资源，建议放在对应页面目录下。

## 页面归档说明

- 每个 HTML 页面应为独立完整的文件，包含全部必要的结构和资源引用（或通过相对路径引用 `assets/` 中的公共资源）。
- 页面内容无特定限制，但建议保持清晰的文件命名和目录结构，便于查找和维护。
- 本仓库不针对任何单个域名或网站，归档的页面可能来自不同用途或实验性项目。

## 维护说明

- 欢迎提交新的独立 HTML 页面，请确保文件结构和引用路径正确。
- 如涉及外部资源（如 CDN 库），建议在页面中注明来源或版本。
- 本仓库不承担页面内容的审核责任，维护者保留对不合规内容进行移除的权利。
- 如需删除或更新已有页面，请提交 Pull Request 并简要说明原因。

## 使用

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下对应的 HTML 文件即可预览。
3. 也可通过 GitHub Pages 或其他静态托管服务发布（需自行配置）。
